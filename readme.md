## Synopsis

Nodely is a start skeleton made from NodeJs consisting of ExpressJs, Passport, MongoDB and Mongoose.

## Motivation

Thought about making a starter template that is quite handy to start any project

## API Reference

* NodeJS
* ExpressJs
* Passport
* MongoDB
* Mongoose
